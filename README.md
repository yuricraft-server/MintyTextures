# Minty Textures
A texture pack for Minecraft

# Installation

### Windows (includes updater)
1. Download `installer-windows.zip` from the latest release
2. Extract the file and run `installer.bat`
3. If using unmodded Minecraft in the default location, enable `Use default resource folder`
4. If not, disable `Use default resource folder` and input your resource folder

### Other
1. Download `MintyTextures.zip` from the latest release
2. Move the file to your resource folder

# Updating the textures

**This only works if you installed with the windows installer, otherwise repeat the installation steps.**

To update the textures, navigate to the installation folder and run `MintyTextures-updater.bat`
<br>
This will auto-install the newest release (you will need to re-enable the pack in minecraft options after updating)
